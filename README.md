# QuizBuzz

<h2 align='center'>Project URL ~ <a href="https://arsh.link/quizbuzz">QuizBuzz</a></h2>

## Features:
* 30 Seconds Timer
* Total 5 Questions
* Questions Have 4 MCQ
* Report Generation Based on Scoring
